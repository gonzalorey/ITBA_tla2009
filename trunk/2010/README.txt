================== Compilacion =======================

1. Descomprimir todos los archivos en la misma carpeta
   respetando el orden de directorios que posee el
   archivo comprimido.
2. Ejecutar "make all" en el directorio del los codigos 
   fuente.

3. Limpieza: Para borrar binarios y archivos auxiliares
   ejecutar "make clear".

================ Uso ===============================

Ejecucion:

 $> ./parser <archivo origen> <archivo destino>


Ejemplo:

$> ./parser samples/test2.txt samples/solution.txt


Abrir el archivo samples/solution.txt para ver el
resultado del analisis.
